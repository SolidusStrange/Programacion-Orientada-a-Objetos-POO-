```java

import java.util.Scanner;

public class Program {

    public static void main(String[] args){

        Scanner scanner = new Scanner(System.in);

    }

}

```

Ejemplo de un programa que pide al usuario ingresar información, lee el string ingresado por el usuario y luego lo imprime.

```java

import java.util.Scanner;

public class Program {

    public static void main(String[] args){

        // Creamos una herramienta que lee lo que ingresa el usuario y lo llama escaner

        Scanner scanner = new Scanner(System.in);

        // Imprime "Escribe un mensaje: "

        System.out.println("Write a message: ");

        // Lee el string escrito por el usuario y lo asigna

        // A la memoria del programa "String message = (string que fue dado como input)"

        String message = scanner.nextline();

        // Imprime el mensaje escrito por el usuario

        System.out.println(message);

    }

}

```

```java
import java.util.Scanner;

public class Program{
	public static void main(String[] args){
		Scanner reader = new Scanner(System.in);
		System.out.println("Write the first string");
		String first = reader.nextline();
		System.out.println("Write the second string");
		String second = reader.nextline();
		System.out.println("Write the third string");
		
		System.out.println("Last string you wrote was " + third + ", which");
		System.out.println("was preceded by " + second + ".");
		System.out.println("The first string was " + first + ".")
		
		System.out.println("All together: " + first + second + third);
	}
}
```
