Las variables pueden ser consideradas como un contenedor en cual la información dada de un tipo puede ser almacenada. Ejemplos de estos diferentes tipos incluyen texto(String), números enteros(int), números decimales(double), and cuando algo puede ser falso o verdadero(boolean). Un valor es asignado a una variable asignando el signo igual(=).
```java
int months = 12;
```
Ejemplo de concatenación y uso de variables:
```java
String text = "contains text";
int wholeNumber = 123;
double floatingPoint = 3.141592653;
boolean trueOrFalse = true;

System.out.println("Text variable: " + text);
System.out.println("Integer variable: " + wholeNumber);
System.out.println("Floating-point variable: " + floatingPoint);
System.out.println("Boolean: " + trueOrFalse);
```

**Ejercicio**: Varias variables
![Descripción opcional](imagenes/Pasted%20image%2020251102112741.png)
```java
package practica;
import java.util.Scanner;
public class Practica {
    public static void main(String[] args) {
        Scanner reader = new Scanner(System.in);

        System.out.println("Chicken: ");
        int chicken = reader.nextInt();
        System.out.println(chicken);

        System.out.println("Bacon (kg): ");
        double bacon = reader.nextDouble();
        System.out.println(bacon);

        // Limpiamos buffer cuando usamos dos tipos distintos de input
        reader.nextLine();

        System.out.println("Tractor: ");
        String tractor = reader.nextLine();
        System.out.println(tractor);

        System.out.println("And finally, a summary:");
        System.out.println(chicken);
        System.out.println(bacon);
        System.out.println(tractor);
    }
}
```
Las variables son únicas y se crea la primera vez que es declarada, por lo que en el siguiente dará un error de: *variable pi is already defined in method main*
```java
public class Example {
    public static void main(String[] args) {
        double pi = 3.14;
        double pi = 3.141592653;

        System.out.println("The value of pi is: " + pi);
    }
}
```
Diferente cuando a una variable ya declarada se le asigna un valor distinto:
```java
int value = 10;
System.out.println(value);
value = 4;
System.out.println(value);
```
```java
int number = 123;
System.out.println("The value of the variable is " + number);

number = 42;
System.out.println("The value of the variable is " + number);
```
Hay que tener en consideración también que el tipo de variable persiste:
```java
boolean integerAssignmentWillWork = false;
integerAssignmentWillWork = 42; // Does not work

int value = 10;
integerAssignmentWillWork = value; // Neither does this
```

Tipos de variables permitidas y no permitidas:
- Permitidas
	- lastDayOfMonth = 20
	- firstYear = 1952
	- name = "Essi"
- No permitidas:
	- last day of month = 20
	- 1day = 1952
	- beware! = 1910
	- 1920 = 1
---

| Type                                  | Example                 | Accepted values                                                                                                                                                                                                                                                                                                          |
| ------------------------------------- | ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Whole number, i.e., `int`             | `int value = 4;`        | Un entero puede contener números enteros cuales valores estén entre los rangos de -2147483648 y 2147483647.                                                                                                                                                                                                              |
| Floating-point number, i.e., `double` | `double value = 4.2;`   | Los números con punto flotante contienen números decimales con el valor el máximo valor posible de 2^1023. Cuando un número es representado con un número con punto flotante, el valor puede ser inexacto ya que los punto-flotante son incapaces de representar todos los números decimales. Esta razón aún se explora. |
| `String`                              | `String text = "Hi!";`  | Un string puede contener texto. Los strings se encierran en comillas.                                                                                                                                                                                                                                                    |
| True or false value, i.e., `boolean`  | `boolean right = true;` | Un boolean puede contener ya se un valor `true` o `false`.                                                                                                                                                                                                                                                               |

| ------- | -------------- |

El otro sería convertir los strings al tipo que queremos usar:

| Tipo    | Funcion                   |
| ------- | ------------------------- |
| int     | Integer.valueOf(variable) |

| boolean | Boolean.valueOf(variable) |
##### Resumen Tipos de Variables

``` java
package practica;
import java.util.Scanner;
public class Practica {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        String text = scanner.nextLine();
        int integer = Integer.valueOf(scanner.nextLine());
        double floatingPoint = Double.valueOf(scanner.nextLine());
        boolean trueOrFalse = Boolean.valueOf(scanner.nextLine());
    }
}
```

``` java
package practica;
import java.util.Scanner;
public class Practica {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("Give a string: ");
        String text = scanner.nextLine();
        System.out.println("Give an integer: ");
        int num = scanner.nextInt();
        System.out.println("Give a double: ");
        // Limpiamos buffer
        scanner.nextLine(); 
        double decim = scanner.nextDouble();
        System.out.println("Give a boolean: ");
        boolean booleanEx = scanner.nextBoolean();
        
        System.out.println("You gave the string " + text);
        System.out.println("You gave the integer " + num);
        System.out.println("You gave the double " + decim);
        System.out.println("You gave the boolean " + booleanEx);
    }
}
```
