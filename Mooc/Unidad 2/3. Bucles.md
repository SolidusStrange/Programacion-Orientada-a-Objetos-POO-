```java
public class Example {
    public static void main(String[] args) {
        int number = 1;

        while (number < 6) {
            System.out.println(number);
            number++;
        }
    }
}
```
Aquí se puede leer como Mientras el valor de la variable sea menor que 6, imprime su valor y súmale uno.
Por lo tanto, aprendimos cómo un bucle `while` con una condición puede ser usado para recorrer números en un intervalo determinado.
```java
int i = 0;
while (i < 10) {
    System.out.println(i);
    i++;
}
```

Lo mismo puede ser logrado usando un bucle `for`:
```java
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}
```

Un bucle `for` contiene cuatro partes:
1. Introducir la variable para contar el número de ejecuciones
2. La condición del bucle
3. Incrementar (o disminuir) el valor de la variable contador
4. La funcionalidad a ejecutar

```java
for (*introducing a variable*; *condition*; *increasing the counter*) {
    // Functionality to be executed
}
```

Se vería así la ejecución:
```java
public class Example {
    public static void main(String[] args) {
        for (int i = 0; i < 5; i++) {
           System.out.println(i);
        }   
    }
}
```
También se puede definir el intervalo usando variables:
```java
int start = 3;
int end = 7;
for (int i = start; i < end; i++) {
    System.out.println(i);
}
```

### Parar la ejecución de un loop

```java
int result = 0; // Inicializa la variable 

int i = 0; // Inicializa el contador
while (true) { 
    result += 3; // shorthand for result = result + 3
    i++;  // shorthand for i = i + 1

    if (i == 4) { // Si es igual a 4 el contador, se para
        break;
    }
}

System.out.println(result); // Se muestra el resultado una vez se acaba el loop
```

También se puede realizar de esta manera sin `break` ni `True` en el while, simplemente  creando una condición.
```java
int result = 0;

int i = 0;
while (i < 4) {
    result += 3; // shorthand for result = result + 3
    i++;  // shorthand for i = i + 1

}

System.out.println(result);
```
Por otro lado, también se puede realizar un bucle con `for` alternando el resultado en cada iteración
```java
int result = 0;

for (int i = 0; i < 4; i++) {
    result += 3;
}

System.out.println(result);
```
